# Marketing Campaign Performance Analysis

## Project Overview

This project analyzes marketing campaign performance using Python and Power BI. The objective is to identify campaign effectiveness, evaluate key marketing KPIs, and build an interactive dashboard that supports data-driven business decisions.

The project covers the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), KPI selection, dashboard development, and business insights.

---

## Business Questions

This project aims to answer the following questions:

1. Which campaign type delivers the highest average ROI?
2. Which marketing channel achieves the highest conversion rate?
3. Which company performs the best?
4. How does marketing performance change over time?
5. Which target audience has the highest engagement?
6. Does campaign performance differ across locations?

## Key Insights

- Search Campaign achieved the highest average ROI.
- Email produced the highest average Conversion Rate.
- ROI remained relatively stable throughout the observation period.
- Campaign performance across locations and target audiences was relatively similar.

---

## Business Recommendations

- Prioritize Search Campaign due to its higher ROI.
- Optimize Email Marketing to improve conversions.
- Evaluate additional factors such as Acquisition Cost and Customer Segment for deeper analysis.
